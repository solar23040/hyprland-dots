# Hyprland Dots

My personal Hyprland + Wayland rice config.

## Quick Install
git clone https://github.com/solar23040/hyprland-dots.git
cd hyprland-dots
./install.sh

## What's included

- **Hyprland** - Wayland compositor config
- **Waybar** - Status bar with Gruvbox theming
- **Alacritty** - Terminal
- **Rofi** - Application launcher
- **Swaylock** - Screen locker
- **Fastfetch** - System info

## Restore backup

If something breaks:
cp -r ~/.config-backup-* ~/.config


## Dependencies

sudo pacman -S hyprland waybar alacritty rofi swaylock fastfetch
